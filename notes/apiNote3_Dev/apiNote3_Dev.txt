# некоторые правила для кострукторов

	function Constructor(...) constructor
	
	1) От 0 аргументов
	Constructor()
	
	2) От одного аргумента типа __ApiWrap
	Constructor(v: __ApiWrap)
	
	3) От одного и более аргументов
	Constructor(...)
	
	Специальные методы:
	
	1) toArray()		-> если можно преобразовать в массив
	2) toClone(f, data) -> клонирование контейнера с обработкой каждого элемента контейнера
		- необходимо, чтобы работало с 0-ом аргументов
